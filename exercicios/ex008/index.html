Tipos de Formatação

Negrito/Itálico

Para colocar em negrito é possível usar a tag b

E também é possivel usar a tag strong

Ja para colocar em italico, você pode usar a tag i

E também a tag em

Texto marcado

Para colocar como texto marcado basta usar a tag MARK.

Texto grande e pequeno

Para colocar texto grande use a tag BIG e para texto pequeno use a tag SMALL.

Texto Deletado

Para texto deletado use a tag DEL.

Texto inserido

Para texto inserido use a tag INS (sublinhar).

Texto sobrescrito

Exemplo:x20+156 use a tag SUP.

Texto subscrito

Exemplo: h2o h2co3 use a tag SUB.
